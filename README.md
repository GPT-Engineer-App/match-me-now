# match-me-now

Create a dating app where people will be matched based on their profiles automatically. They will have some attributes and preferences like hobbies, interests etc and will be matched based on those. All other functionalities will be like Tinder let's say.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/match-me-now.git
cd match-me-now
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
